# Нейрободр - Мониторинг состояния водителей

## Что это?
Программа для регистрации и авторизации водителей. Данные сохраняются в CSV файл.

## Как работает?
**Регистрация:**
1. Вводишь фамилию, имя, отчество, возраст (Ураганов Ураган Ураганович, 18)
2. Загружаешь фото (обязательно 300x300 пикселей, uragan.png)
3. Нажимаешь "Записать"

**Авторизация:**
1. Вводишь логин и пароль
2. Происходит проверка логина и пароля
3. Если всё верно - запускается мониторинг

## Как запустить?
```bash
pip install pillow
python идентификация.py
